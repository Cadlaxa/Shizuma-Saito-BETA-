# Arpasing-Voicebank
this voicebank uses my ver 2 ARPAsing reclist, it consists of 4 pitches `G#2`, `C3`, `F3`, and `A#3`. This voicebank has a complete VV transition, complete CC without [hh_C] and [dx_c], and has extra phonemes.
